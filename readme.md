View and set command aliases in Windows.

### How to setup?
Just run `setup.reg`. After that you can manage your aliases.  
Aliases file is located in `%USERPROFILE%\Documents\Scripts`.

### Managing aliases
Get list of aliases:

    alias
Add an alias:

    alias test=dir \b $*
Remove some alias:

    alias test=
